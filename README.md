# Dynamic Brainfuck interpreter and transpiler

This program implemets a basic interpreter and transpiler for the [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) Programming language.

Usage:

```bash
bf <source> <targets...> [options...]
Supported options:
  -r[un]     - to run the program after compilation
Supported targets:
  -c
  -j[ava]    - tbd
  -py[thon]  - tbd
  -js        - tbd
  -rust      - tbd
  -go
```
